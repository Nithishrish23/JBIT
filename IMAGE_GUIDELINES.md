# Image and Banner Guidelines

## Banner Sizes
For optimal display across devices, please adhere to the following dimensions:

- **Desktop Main Banner**: 1920 x 600 px
- **Tablet Banner**: 1024 x 400 px
- **Mobile Banner**: 600 x 300 px
- **Ad Sidebar / Skyscraper**: 300 x 600 px
- **Footer / Secondary Banner**: 1200 x 200 px

## Product Images
- **Main Product Image**: 800 x 800 px (Square, White background recommended)
- **Thumbnail**: 200 x 200 px
- **Zoom View**: 1600 x 1600 px

## Profile & Logos
- **User Profile**: 400 x 400 px (Circular crop)
- **Shop Logo**: 500 x 500 px
- **Favicon**: 32 x 32 px or 16 x 16 px

## File Formats
- **Photos**: JPG or WEBP (for better compression)
- **Graphics/Logos**: PNG or SVG (Transparent background)
- **Max File Size**: 2MB per image (Recommended < 500KB for web performance)
